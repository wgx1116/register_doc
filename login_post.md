# 登录

## 说明

|  Auth  |  Method  |  url  |
| :----  | :----:   | :---- |
|  NONE  |  POST  |  /login  |

## 请求参数

|  参数  |  类型  |  是否必选  |  说明  |
| :---- | :----: | :----:   | :----  |
| phone | string | 1 | 手机号码 |
| code | string | 1 | 手机短信验证码 |

## 响应参数

|  参数  |  类型  |  是否必选  |  说明  |
| :---- | :----: | :----:   | :----  |

## 请求示例

```
{
    "phone": "15623967785",
    "code": "592884"
}
```

## 响应示例

```
{
    "code": 200,
    "msg": "success"
}
```
